# StackPay

StackPay is a revolutionary stablecoin payment platform built on the Stacks blockchain. It leverages the security and stability of Bitcoin through the Stacks layer to ensure safe transactions. By using USDCx, users can enjoy the benefits of a stable digital currency for their daily needs.

## Features

### Send & Receive USDCx
Our platform allows you to seamlessly send and receive USDCx across the Stacks network with minimal fees. Experience near-instant settlement times for all your transfers, ensuring your money is where it needs to be when you need it.
- **Utility Bill Payments**: Pay for airtime, data, electricity bills, and cable TV using USDCx.

## Technology

- **Blockchain**: Stacks
- **Currency**: USDCx (Stablecoin on Stacks)
