# StackPay

StackPay is a revolutionary stablecoin payment platform built on the Stacks blockchain.

## Features

- **Send & Receive USDCx**: Seamless stablecoin transactions.
- **Utility Bill Payments**: Pay for airtime, data, electricity bills, and cable TV using USDCx.

## Technology

- **Blockchain**: Stacks
- **Currency**: USDCx (Stablecoin on Stacks)
